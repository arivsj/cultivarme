# cultivarme
Uma ferramenta simples e eficaz para ajudar no desenvolvimento da autoconfiança, focada em privacidade total e acessibilidade universal.

Como desenvolvedor #AndroidNativo que respira #Kotlin, meu DNA me puxava para criar um app nativo. Cheguei a considerar #ReactNative para multiplataforma, ou até uma stack completa com #React, #NextJS e #NodeJS...

Mas quando o assunto é autoconfiança, complexidade técnica pode ser uma barreira. Então, fiz uma escolha consciente: a simplicidade como feature principal.

Desconstruí tudo e fui para o básico universal: #HTML, #CSS e #JavaScript puro!  Um único arquivo. Sem instalação. Sem complicação.

Os 3 pilares que guiaram essa decisão:

PRIVACIDADE TOTAL: Seus dados são só seus! O app não tem backend, não coleta nada. Tudo fica salvo localmente. Quer mais segurança? Baixe e rode localmente! 

ACESSO SEM BARREIRAS: Funciona em qualquer dispositivo com navegador. Do flagship ao básico.

LIVRE E ABERTO: É e sempre será gratuito. E por ser #OpenSource, pode evoluir com a comunidade. 

Às vezes, a solução mais poderosa não está na stack mais complexa, mas na que resolve o problema com elegância para o maior número de pessoas.
